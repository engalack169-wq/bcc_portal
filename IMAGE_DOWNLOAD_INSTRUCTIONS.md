# ЁЯЖШ Troubleshooting

### Issue: Downloaded files are too large
**Solution:** Use TinyPNG to compress
- Go to: https://tinypng.com
- Drag & drop your image
- Keep compressed version
- Max size should be ~200KB per image

### Issue: Can't find Download button
**Solution:** Try these
- Make sure you clicked on the image (opens full preview)
- Download button usually in top-right corner
- If not visible, right-click image тЖТ "Save image as"
- Choose location manually

### Issue: Image quality looks bad
**Solution:** 
- Download "Original" size (not thumbnail)
- Most search results show small previews
- Click to see full resolution
- Unsplash defaults to high quality

### Issue: Name doesn't match exactly
**Solution:**
- Right-click file тЖТ Rename
- Paste exact name from checklist
- Include `.jpg` extension
- Don't use spaces or special characters
